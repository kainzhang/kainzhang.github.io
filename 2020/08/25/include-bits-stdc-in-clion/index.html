解决MacOS Clion引用 <bits/stdc++.h> 问题

  MacOS默认使用Clang，没有bits/stdc++.h，需要手动添加

在 Clion -> Preferences -> Build -> Toolchains 查看 C++ Compiler 的路径，如下图所示。

cd 至 /usr 即可，在该目录下找到include文件夹，在该文件夹内创建 bits 文件夹，然后把下面的 stdc++.h 拖入文件夹即可。
